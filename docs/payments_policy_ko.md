# 결제 및 환불 정책 — Payments Policy (한국어)

**버전**: 1.0.0
**최종 업데이트**: 2025-11-04
**담당**: 재무운영팀 (Finance Operations)
**상태**: Active

---

## 1. 개요

### 1.1 목적

본 정책은 DecisionOS 플랫폼의 결제, 환불, 분쟁 처리에 대한 기준을 정의합니다:
- **결제 수단** 및 승인 절차
- **환불** 및 조정 정책
- **분쟁 및 차지백** 대응 절차
- **세금 및 부가세** 처리
- **개인정보** 및 보안 원칙

**적용 범위**: 모든 DecisionOS 유료 서비스 이용 고객

---

### 1.2 용어 정의

| 용어 | 정의 |
|------|------|
| **결제 의도 (Payment Intent)** | 결제를 시작하기 위한 서버 측 객체 (금액, 통화, 고객 정보 포함) |
| **승인 (Authorization)** | 카드사/PG로부터 결제 가능 여부 확인 (실제 출금 전) |
| **매입 (Capture)** | 승인된 금액을 실제로 고객 계좌에서 출금 |
| **환불 (Refund)** | 매입된 금액의 전부 또는 일부를 고객에게 반환 |
| **차지백 (Chargeback)** | 고객이 카드사를 통해 직접 결제 취소를 요청하는 분쟁 |
| **정산 (Settlement)** | PG사가 가맹점(DecisionOS)에 실제 입금하는 과정 |
| **KYC (Know Your Customer)** | 고객 신원 확인 절차 (본인 인증, 사업자 등록증 확인 등) |

## 2. 결제 수단 및 방법

### 2.1 지원 결제 수단

**개인 고객**:
- ✅ 신용카드 (Visa, MasterCard, AMEX)
- ✅ 체크카드 (국내 발급 은행)
- ✅ 간편결제 (네이버페이, 카카오페이, 토스) — 향후 지원 예정
- ❌ 현금, 수표 — 미지원

**사업자 고객**:
- ✅ 법인카드
- ✅ 계좌이체 (세금계산서 발행 시)
- ✅ 후불 결제 (신용 심사 통과 시, 월말 정산)

---

### 2.2 결제 프로세스

**1단계: 인보이스 발행**
```
사용량 집계 → 인보이스 생성 (월 1일 자동) → 이메일 발송
```

**2단계: 결제 시도**
```
[자동 결제 설정 시]
등록된 결제 수단으로 자동 결제 시도 (월 3일, 5일, 7일 재시도)

[수동 결제 시]
고객이 인보이스에서 "지금 결제" 클릭 → 결제 페이지 이동 → 결제 완료
```

**3단계: 승인 및 매입**
```
승인(Authorization): 카드사 승인 확인 (즉시)
매입(Capture): 승인 후 1~2일 이내 실제 출금
```

**4단계: 영수증 발행**
```
매입 완료 → 영수증 PDF/JSON 생성 → 이메일 발송 + 포털에서 다운로드 가능
```

---

### 2.3 결제 실패 시 처리

**자동 재시도**:
- 1차 실패: 2일 후 재시도 (월 5일)
- 2차 실패: 2일 후 재시도 (월 7일)
- 3차 실패: 서비스 일시 정지 경고 (이메일 + SMS)

**서비스 정지**:
- 최종 재시도 실패 후 7일 경과: 서비스 일시 정지
- 정지 후 30일 경과: 계정 비활성화 (데이터 백업 보관, 90일 후 삭제 예정 통지)

**복구**:
- 고객이 미납 금액 결제 → 즉시 서비스 재개
- 연체료: 없음 (단, 법적 청구 시 법정 이자율 적용 가능)

## 3. 환불 정책

### 3.1 환불 가능 사유

**전액 환불**:
- ✅ 서비스 장애로 인한 SLA 미달 (계약서 명시된 환불 조건 충족 시)
- ✅ 중복 결제 (동일 인보이스에 대해 2회 이상 결제된 경우)
- ✅ 오류 청구 (시스템 오류로 과다 청구된 경우)
- ✅ 계약 해지 시 미사용 기간 (선불 요금제 한정, 일할 계산)

**부분 환불**:
- ✅ 일부 기능 미제공 (SLA 보상 조항에 따라 크레딧 또는 현금 환불)
- ✅ 다운그레이드 (상위 요금제 → 하위 요금제, 차액 환불)

**환불 불가**:
- ❌ 단순 변심 (서비스를 실제 사용한 경우)
- ❌ 이용약관 위반으로 인한 계정 정지
- ❌ 무료 체험 기간 (환불 대상 아님)

---

### 3.2 환불 신청 및 처리

**신청 방법**:
1. 고객 지원 포털에서 "환불 요청" 제출
2. 사유, 인보이스 번호, 환불 금액 명시
3. 증빙 자료 첨부 (해당 시, 예: 중복 결제 내역 스크린샷)

**처리 절차**:
```
신청 접수 → 검토 (영업일 기준 3일 이내) → 승인/거부 통지
→ 승인 시: 환불 처리 (5~10영업일 소요)
→ 거부 시: 사유 안내 + 이의 신청 경로 제공
```

**환불 방법**:
- **원결제 수단으로 환불** (신용카드 → 카드 취소, 계좌이체 → 원계좌 입금)
- 원결제 수단 사용 불가 시: 고객 지정 계좌로 입금

**환불 수수료**:
- 회사 귀책 사유: **수수료 없음**
- 고객 귀책 사유: **환불 금액의 3%** (최소 5,000원) 공제 후 환불

---

### 3.3 환불 예외 및 특수 케이스

**크레딧 환불** (현금 환불 대신):
- 소액 환불 (<10,000원): 현금 대신 서비스 크레딧 제공 가능 (고객 동의 시)
- 크레딧 유효기간: 발급일로부터 12개월

**부분 사용 후 환불**:
- 월 중간 해지 시: 일할 계산 (사용 일수 / 전체 일수 × 월 요금)
- 예: 15일 사용 후 해지 → 50% 환불 (30일 기준 월)

**프로모션/할인 적용 건**:
- 환불 시 할인 전 정가 기준으로 사용량 차감 후 환불
- 예: 정가 100,000원을 50% 할인받아 50,000원 결제 → 50% 사용 시 환불액 = 25,000원 (할인 후 금액의 50%)

## 4. 세금 및 부가세

### 4.1 부가가치세 (VAT)

**적용 대상**:
- 국내 사업자: **10% VAT** 부과 (세금계산서 발행)
- 개인 고객: **10% VAT** 포함 (현금영수증 또는 카드 전표)

**면세 대상**:
- 해외 고객 (국외 거래, VAT 면제)
- 교육 기관 (비영리, 승인된 경우만)

**세금계산서**:
- 발행 시기: 매입 완료 익월 10일까지
- 발행 방법: 전자세금계산서 (국세청 승인 시스템)
- 조회: 고객 포털 → "세금계산서" 메뉴에서 다운로드

---

### 4.2 원천징수 및 기타 세금

**해외 거래 (역외 서비스)**:
- 한국 내 사업자가 해외 고객에게 제공: 영세율 (0%) 적용 가능 (수출 실적 증빙 필요)

**원천징수**:
- 국내 개인 고객 중 사업소득 인정 시: 3.3% 원천징수 (해당 시)

## 5. 분쟁 및 차지백

### 5.1 분쟁 (Dispute) 절차

**분쟁 발생 상황**:
- 고객이 결제 내역에 대해 이의 제기
- 사유: "서비스 미제공", "금액 오류", "비인가 결제" 등

**처리 절차**:
```
고객 → 카드사에 분쟁 제기
카드사 → DecisionOS에 분쟁 통지 (Webhook)
DecisionOS → 증빙 자료 제출 (사용 내역, 계약서, 로그)
카드사 → 판정 (고객 승 / 가맹점 승)
```

**대응 기한**:
- 분쟁 통지 수신 후 **7일 이내** 증빙 제출 (기한 초과 시 자동 패소)

**증빙 자료**:
- 서비스 사용 로그 (API 호출 기록, 대시보드 접속 기록)
- 인보이스 및 이용약관 동의 내역
- 고객과의 이메일/채팅 기록

---

### 5.2 차지백 (Chargeback)

**차지백 발생**:
- 고객이 카드사를 통해 직접 결제 취소 요청 (DecisionOS 거치지 않음)

**차지백 수수료**:
- 카드사 부과: 건당 **15,000~30,000원** (PG사 정책에 따라 다름)
- 패소 시 이 수수료는 DecisionOS 부담

**방어 전략**:
- **사전 예방**: 명확한 서비스 설명, 인보이스 이메일 발송, 환불 정책 고지
- **증빙 강화**: 모든 거래에 대해 감사 추적 가능한 로그 유지
- **신속 대응**: 분쟁 통지 즉시 HITL 케이스 생성, 전담 팀 배정

---

### 5.3 사기 및 비인가 결제

**비인가 결제 감지**:
- PG사 사기 탐지 시스템 (Fraud Detection)
- 이상 패턴: 짧은 시간 내 다수 결제, 해외 IP에서 국내 카드 사용 등

**대응**:
- 의심 거래 자동 차단 (승인 거부)
- 고객에게 본인 확인 요청 (SMS, 이메일 인증)
- 확인 완료 후 수동 승인

**고객 보호**:
- 비인가 결제 확인 시 **즉시 전액 환불** (고객 부담 없음)
- 카드 정보 갱신 요청 (토큰 무효화)

## 6. 개인정보 및 보안

### 6.1 카드 정보 보안

**PCI-DSS 준수**:
- 카드 번호 **미저장** (PG사 토큰만 저장)
- 저장 데이터: 마지막 4자리, 카드 브랜드 (Visa/MasterCard), 만료 월/년
- 전체 카드 번호는 PG사 서버에만 존재

**토큰화**:
```
고객 입력: 1234-5678-9012-3456
PG 토큰: tok_a1b2c3d4e5f6 (일회용 또는 재사용 가능)
저장 정보: 카드 끝자리 3456, Visa, 만료 12/2027
```

---

### 6.2 결제 정보 접근 제한

**권한 관리**:
- **billing_admin**: 인보이스 생성, 결제 수단 등록, 환불 승인
- **finance_ops**: 정산 조회, 대사 리포트
- **auditor**: 읽기 전용 (감사 목적)
- **일반 사용자**: 본인 계정의 인보이스 및 영수증만 조회

**접근 로그**:
- 모든 결제 정보 접근 기록 (IP, 사용자, 타임스탬프, 작업 내용)
- 로그 보존 기간: **7년** (금융 감사 대응)

---

### 6.3 웹훅 보안

**서명 검증**:
- PG사 웹훅 요청에 서명 (HMAC-SHA256) 포함
- DecisionOS는 서명 검증 후에만 이벤트 처리 (위조 방지)

**재생 공격 방지**:
- 타임스탬프 허용 범위: ±5분 (오래된 요청 거부)
- 이벤트 ID 중복 확인 (동일 이벤트 재처리 방지, 멱등성)

## 7. 정산 및 대사

### 7.1 정산 주기

**PG사 → DecisionOS 입금**:
- 주기: **D+3 영업일** (매입일 기준 3일 후 입금)
- 예: 11월 1일 매입 → 11월 6일 입금 (주말 제외)

**정산 내역**:
- PG사에서 제공하는 정산 파일 (CSV/Excel)
- 포함 정보: 거래 ID, 매입 금액, 수수료, 순 입금액

---

### 7.2 대사 (Reconciliation)

**목적**: 내부 원장 vs PG 정산 파일 일치 확인

**절차**:
```
1. 정산 파일 다운로드 (매일 자동 수집)
2. 내부 원장(ledger_txns)과 매칭
3. 불일치 건 추출 → 대사 리포트 생성
4. 불일치 사유 분석:
   - PG사 미기재 (누락) → PG사 확인 요청
   - 내부 기록 오류 → 수동 조정
5. 조정 완료 → 원장 최종 마감
```

**허용 오차**:
- 금액 불일치: **±100원** (환율/단수 차이 허용)
- 건수 불일치: **0건** (모든 거래 매칭 필수)

**불일치율 목표**:
- **≤ 0.5%** (월 1,000건 거래 시 최대 5건 불일치 허용)

- ❌ "연체" → "미결제 내역"

## 9. 자주 묻는 질문 (FAQ)

### Q1. 인보이스는 언제 발행되나요?
**A**: 매월 1일 자동 발행됩니다. 전월(1~말일) 사용량 기준으로 산정됩니다.

### Q2. 결제는 언제 이루어지나요?
**A**: 자동 결제 설정 시 **월 3일**에 첫 시도, 실패 시 5일, 7일 재시도합니다. 수동 결제는 언제든 가능합니다.

### Q3. 결제 수단을 변경하고 싶어요.
**A**: 고객 포털 → "결제 수단 관리" → "새 카드 추가" 또는 "기본 결제 수단 변경"

### Q4. 중복 결제되었어요. 어떻게 하나요?
**A**: 고객 지원팀에 문의 주시면 확인 후 **즉시 환불** 처리됩니다 (영업일 기준 3일 이내).

### Q5. 세금계산서는 어디서 받나요?
**A**: 매입 완료 익월 10일까지 이메일로 발송되며, 포털 → "세금계산서" 메뉴에서 다운로드 가능합니다.

### Q6. 환불은 얼마나 걸리나요?
**A**: 승인 후 **5~10영업일** 소요됩니다 (카드사 처리 기간 포함).

### Q7. 해외 카드로 결제 가능한가요?
**A**: Visa, MasterCard, AMEX 국제 카드는 가능합니다. 단, 환율 및 해외 결제 수수료는 카드사 정책에 따릅니다.

### Q8. 계좌이체로 결제하고 싶어요.
**A**: 사업자 고객에 한해 가능합니다. 고객 지원팀에 문의하시면 계좌 정보를 안내해 드립니다.

### Q9. 서비스를 해지하면 환불 받을 수 있나요?
**A**: 선불 요금제의 경우 미사용 기간에 대해 일할 계산 후 환불 가능합니다. 후불 요금제는 사용한 만큼만 청구됩니다.

### Q10. 차지백이 뭔가요?
**A**: 고객이 카드사를 통해 직접 결제 취소를 요청하는 절차입니다. 분쟁이 발생하면 양측 증빙을 검토 후 카드사가 판정합니다.

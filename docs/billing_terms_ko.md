# 청구 약관 — Billing Terms (한국어)

**버전**: 1.0.0
**최종 업데이트**: 2025-11-03
**담당**: 재무운영팀
**상태**: Active

---

## 1. 청구 주기

### 1.1 월별 청구 (Pro 플랜)

**청구일**: 매월 1일 (전월 1일~말일 사용량 기준)
**발송**: 이메일로 청구서(Invoice) 발송
**결제 기한**: 청구서 발행 후 3일 이내
**자동 결제**: 등록된 카드로 자동 결제 (월 3일)

**예시**:
```
11월 1일~30일 사용
12월 1일: 청구서 발행 (INV-2025-11-00123)
12월 3일: 자동 결제 시도
12월 4일: 결제 기한 (미결제 시 재시도)
```

---

## 2. 청구 항목

### 2.1 기본 요금

**Pro 플랜**: ₩100,000/월 (VAT 별도)
- 포함 내역: 5,000 decision calls/day, 200 GB storage, 10 users

### 2.2 초과 사용 요금

| 항목 | 포함 | 초과 요금 |
|------|------|----------|
| Decision Calls | 5,000/day | ₩2/call |
| Storage | 200 GB | ₩80/GB/month |
| HITL Cases | 500/month | ₩500/case |
| 추가 사용자 | 10명 | ₩10,000/명/month |
| 추가 프로젝트 | 10개 | ₩5,000/개/month |

**계산 예시**:
```
기본 요금: ₩100,000
초과 Decision Calls (2,000개 × ₩2): ₩4,000
초과 Storage (50 GB × ₩80): ₩4,000
───────────────────────────────────
소계: ₩108,000
부가세 (10%): ₩10,800
───────────────────────────────────
합계: ₩118,800
```

## 3. 결제 방법

**지원 수단**:
- ✅ 신용카드 (자동 결제)
- ✅ 법인카드
- ✅ 계좌이체 (사전 승인 필요)

**카드 등록**: 고객 포털 → 결제 수단 관리

---

## 4. 결제 실패 시 처리

**재시도 일정**:
- 1차 실패 (3일): 2일 후 재시도 (5일)
- 2차 실패 (5일): 2일 후 재시도 (7일)
- 3차 실패 (7일): 서비스 일시 정지 경고

**서비스 정지**:
- 최종 재시도 실패 후 7일 경과: 서비스 정지
- 정지 후 30일 경과: 데이터 삭제 예정 통지

---

## 5. 환불 및 조정

**환불 가능 사유**:
- ✅ 서비스 장애로 인한 SLA 미달
- ✅ 과다 청구 (시스템 오류)
- ✅ 중복 결제

**환불 불가**:
- ❌ 단순 변심 (이미 사용한 서비스)
- ❌ 약관 위반으로 인한 계정 정지

**환불 신청**: 고객 지원 포털 → 환불 요청

---

## 6. 플랜 변경

### 6.1 업그레이드

**Free → Pro**:
- 즉시 적용
- 일할 계산 (남은 기간만 청구)

**예시**:
```
11월 15일 업그레이드 (15일 남음)
청구: ₩100,000 × (15/30) = ₩50,000
```

### 6.2 다운그레이드

**Pro → Free**:
- 현재 청구 주기 종료 시 적용 (즉시 적용 아님)
- 환불 없음 (남은 기간 사용 가능)

**데이터 보존**:
- 다운그레이드 후 30일: 데이터 read-only (초과분 삭제 요청)
- 30일 경과: 자동 보관 처리

# Appeals Response Templates (Korean)

## 목적
이의 제기(Appeals)에 대한 표준화된 응답 템플릿으로, 고객에게 명확하고 공정하며 존중하는 커뮤니케이션을 제공합니다.

---

## 일반 가이드라인

### 작성 원칙
1. **명확성**: 결정과 근거를 이해하기 쉽게 설명
2. **공정성**: 객관적이고 편견 없는 평가
3. **존중**: 고객의 상황과 감정을 이해하고 배려
4. **투명성**: 결정 프로세스와 근거를 투명하게 공개
5. **건설성**: 개선 방안과 다음 단계를 제시

### 피해야 할 표현
❌ **부정적/공격적 표현**:
- "귀하의 주장은 근거가 없습니다"
- "불가능합니다"
- "규정상 안 됩니다"
- "이미 결정되었습니다"

✅ **대안 표현**:
- "제출하신 증빙을 검토한 결과..."
- "현재 기준으로는 어려우나, 다음 조건 충족 시 가능합니다"
- "관련 정책에 따르면..."
- "재검토 결과..."

### 민감 어휘 가이드
**신용 관련**:
- ❌ "신용불량", "블랙리스트" → ✅ "신용 평가 결과", "신용 이력"
- ❌ "리스크가 높다" → ✅ "추가 검토가 필요합니다"
- ❌ "부적격" → ✅ "현재 기준 충족이 어렵습니다"

**소득 관련**:
- ❌ "소득이 부족하다" → ✅ "최소 소득 기준에 미달합니다"
- ❌ "증빙이 불충분하다" → ✅ "추가 소득 증빙이 필요합니다"

**거부 관련**:
- ❌ "거부되었습니다" → ✅ "승인이 어렵습니다", "현재로서는 진행이 어렵습니다"

## Template 1: Uphold (유지) - 새로운 증빙 불충분

### 제목
```
이의 제기 검토 결과 안내 - [신청번호]
```

### 본문

```markdown
{고객명} 고객님께,

먼저 저희 서비스에 관심을 가져주시고 소중한 시간을 내어 이의를 제기해 주신 점 감사드립니다.

## 검토 결과

고객님께서 제출하신 이의 제기 건(접수번호: {appeal_id})에 대해 신중하게 재검토를 진행하였습니다.
제출하신 추가 자료와 설명을 면밀히 검토한 결과, 아쉽게도 **원래의 결정을 유지**하게 되었음을 알려드립니다.

## 유지 사유

재검토 결과, 다음과 같은 이유로 현재로서는 승인이 어렵다고 판단되었습니다:

### 1. {주요 사유 1}
{상세 설명}

**현재 상황**:
- {구체적 수치나 상태}

**기준**:
- {승인 기준}

**차이**:
- {기준과의 차이}

### 2. {주요 사유 2}
{상세 설명}

### 3. {주요 사유 3} (해당 시)
{상세 설명}

## 제출하신 추가 자료 검토

고객님께서 제출하신 {서류명}을 확인하였습니다:

- **{서류 1}**: {검토 결과 및 영향}
- **{서류 2}**: {검토 결과 및 영향}

제출하신 자료는 {긍정적 측면}을 확인하는 데 도움이 되었습니다.
그러나 {여전히 부족한 부분}으로 인해 현재 승인 기준을 충족하기는 어려운 상황입니다.

## 재신청 안내

향후 다음 조건이 개선되시면 재신청을 환영합니다:

### 개선 필요 사항
1. **{개선 항목 1}**
   - 현재: {현재 상태}
   - 필요: {필요한 수준}
   - 예상 소요 기간: {기간}

2. **{개선 항목 2}**
   - 현재: {현재 상태}
   - 필요: {필요한 수준}
   - 예상 소요 기간: {기간}

3. **{개선 항목 3}** (해당 시)
   - 현재: {현재 상태}
   - 필요: {필요한 수준}
   - 예상 소요 기간: {기간}

### 권장 재신청 시기
위 사항이 개선되면 **{권장 시기}** 이후 재신청하시기를 권장드립니다.

## 대안 상품 안내

현재 상황에 더 적합할 수 있는 다른 상품을 안내드립니다:

- **{대안 상품 1}**: {간단한 설명}
  - 문의: {연락처 또는 링크}

- **{대안 상품 2}**: {간단한 설명}
  - 문의: {연락처 또는 링크}

## 추가 문의

이 결정에 대해 추가 질문이 있으시거나 상황 개선 후 재신청을 원하실 경우,
언제든지 아래로 연락 주시기 바랍니다:

- **고객센터**: 1588-XXXX (평일 9:00-18:00)
- **이메일**: customer-support@company.com
- **온라인 상담**: https://www.company.com/support

고객님의 금융 목표 달성을 진심으로 응원하며,
상황이 개선되어 다시 찾아주실 날을 기다리겠습니다.

감사합니다.
```

### 템플릿 선택 플로차트

```
Appeals 검토 완료
   ↓
결정 유형은?
   ├─ Uphold (유지)
   │    ├─ 새 증빙 불충분 → Template 1
   │    └─ 오류 주장 기각 → Template 6
   │
   ├─ Overturn (번복)
   │    ├─ 새 증빙으로 승인 → Template 2
   │    ├─ 오류 수정 → Template 3
   │    ├─ 정책 변경 → Template 7
   │    └─ 시간 경과/개선 → Template 8
   │
   └─ Partial (조건부)
        ├─ 담보/보증 필요 → Template 4
        └─ 한도 축소 → Template 5
```

### 맞춤화 가이드

1. **{변수} 교체**: 모든 중괄호 변수를 실제 값으로 교체
2. **섹션 조정**: 해당 없는 섹션은 삭제, 필요한 섹션은 추가
3. **톤 조정**: 고객 상황에 따라 톤 조정 (사과 강화, 축하 강조 등)
4. **길이 조정**: 복잡도에 따라 섹션 상세도 조정

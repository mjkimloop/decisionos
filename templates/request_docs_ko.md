# Request Documents Templates (Korean)

## 목적
케이스 검토 중 추가 서류가 필요할 때 사용하는 표준화된 요청 템플릿입니다.

---

## 일반 가이드라인

### 작성 원칙
1. **명확성**: 필요한 서류와 이유를 명확히 설명
2. **구체성**: 서류 종류, 형식, 제출 방법 구체적 안내
3. **기한**: 합리적인 제출 기한 제시 (일반적으로 3-7일)
4. **지원**: 어려움 시 지원 방법 안내

### 피해야 할 표현
❌ "서류를 제출하세요" (명령조)
✅ "다음 서류를 제출해 주시면 검토를 진행하겠습니다" (요청조)

❌ "불충분합니다" (부정적)
✅ "추가 확인이 필요합니다" (중립적)

## Template 1: 소득 증빙 요청 (근로자)

### 제목
```
[{회사명}] 추가 서류 요청 안내 - 소득 확인 [신청번호]
```

### 본문

```markdown
{고객명} 고객님께,

{제품명} 신청 건(신청번호: {application_id})을 검토하고 있습니다.

## 추가 서류 요청

신청 내용을 신중히 검토한 결과, 정확한 심사를 위해 **소득 확인을 위한 추가 서류**가 필요합니다.

### 요청 사유
현재 제출하신 정보만으로는 소득 수준을 충분히 확인하기 어려워,
보다 정확한 평가를 위해 추가 서류를 요청드립니다.

이는 고객님의 상황을 정확히 파악하여 **최적의 조건**을 제공하기 위함입니다.

## 필요 서류

다음 서류 중 **해당하는 모든 서류**를 제출해 주시기 바랍니다:

### 필수 서류

#### 1. 소득 증빙
다음 중 **최소 1가지 이상** 제출:

- **원천징수영수증**
  - 최근 연도 (예: 2024년)
  - 발급 기관: 직장 (인사팀 또는 경리팀)
  - 형식: PDF 또는 이미지 (스캔 또는 사진)

- **소득금액증명원**
  - 최근 연도 (예: 2024년)
  - 발급 기관: 국세청 홈택스 (www.hometax.go.kr)
  - 형식: PDF 원본

- **급여명세서**
  - 최근 **3개월분** (예: 2025년 8월~10월)
  - 회사 직인 또는 전자서명 포함
  - 형식: PDF 또는 이미지

#### 2. 재직 증명
다음 중 **1가지** 제출:

- **재직증명서**
  - 발급일: 최근 **1개월 이내**
  - 발급 기관: 직장 (인사팀)
  - 포함 정보: 입사일, 직급, 재직 상태
  - 형식: PDF 또는 이미지 (회사 직인 포함)

- **건강보험자격득실확인서**
  - 발급일: 최근 **1개월 이내**
  - 발급 기관: 국민건강보험공단 (www.nhis.or.kr)
  - 형식: PDF 원본

#### 3. 급여 입금 내역 (선택 - 제출 시 심사 유리)
- **통장 거래내역**
  - 최근 **3개월분** 급여 입금 내역
  - 은행 앱 또는 인터넷뱅킹에서 다운로드
  - 형식: PDF 또는 이미지
  - 급여 입금 내역이 명확히 표시되어야 함

### 추가 서류 (해당 시)

#### 부업 또는 추가 소득이 있는 경우
- 추가 소득 증빙 (프리랜서 계약서, 임대 소득 증명 등)
- 사업소득이 있다면: 사업자등록증 사본

## 서류 발급 방법

### 원천징수영수증
1. 직장 인사팀 또는 경리팀에 요청
2. 또는 국세청 홈택스에서 조회/출력
   - www.hometax.go.kr
   - 로그인 → My홈택스 → 지급명세서 등 제출내역
   - 근로소득 지급명세서 조회

### 소득금액증명원
1. 국세청 홈택스 접속 (www.hometax.go.kr)
2. 로그인 (공동인증서 또는 간편인증)
3. 민원증명 → 소득금액증명
4. 발급연도 선택 → PDF 다운로드

### 재직증명서
1. 직장 인사팀에 요청
2. 또는 정부24에서 발급 (일부 기업)
   - www.gov.kr
   - 로그인 → 재직증명서 신청

### 건강보험자격득실확인서
1. 국민건강보험공단 사이트 (www.nhis.or.kr)
2. 로그인
3. 민원여기요 → 개인민원 → 자격득실확인서
4. PDF 다운로드
```

```markdown
자영업자로서 정확한 소득 확인을 위해 추가 서류가 필요합니다.

### 요청 사유
자영업 소득은 근로소득과 달리 변동성이 있어, 보다 정확한 소득 확인을 위해

## 필요 서류

### 필수 서류

#### 1. 사업자등록증 사본
- 발급 기관: 국세청
- 형식: PDF 또는 이미지 (명확한 사진)
- 유효한 사업자등록증 (폐업 상태 아님)

#### 2. 소득 증빙 (다음 중 택1 이상)

**옵션 A: 소득금액증명원** (권장)
- 최근 **2개년도** (예: 2023년, 2024년)
- 발급 기관: 국세청 홈택스
- 형식: PDF 원본
- 발급 방법:
  1. www.hometax.go.kr 접속
  2. 로그인 (공동인증서)
  3. 민원증명 → 소득금액증명
  4. 연도별로 발급 (2023년, 2024년 각각)

**옵션 B: 종합소득세 신고서**
- 최근 **2개년도** (예: 2023년, 2024년)
- 국세청 접수 확인 도장 또는 전자접수 확인
- 형식: PDF

**옵션 C: 부가가치세 신고서**
- 최근 **6기** (예: 2023년 1기 ~ 2024년 2기)
- 국세청 접수 확인 도장 또는 전자접수 확인
- 형식: PDF

#### 3. 사업장 현황 (다음 중 택1)

**사업자 현황 증명**
- 발급: 국세청 홈택스
- 최근 1개월 이내 발급
- 사업 상태 확인용

**또는 사업 통장 거래내역**
- 최근 **6개월**
- 매출 입금 내역 확인 가능한 통장
- 형식: PDF 또는 이미지

### 선택 서류 (제출 시 심사 유리)

#### 재무제표 (규모 큰 사업자)
- 최근 2개년도 재무상태표, 손익계산서
- 세무사 확인 또는 법인 등기

#### 임대차 계약서 (사업장 운영 중인 경우)
- 현재 사업장 임대차 계약서
- 사업 안정성 확인용

#### 주요 거래처 계약서
- 지속적 매출이 발생하는 거래처
- 사업 지속 가능성 확인용
```
